# 🎨 TSIS 2 - Paint Application
## 📌 Description
This project is a Paint application created using Python and Tkinter.
The application supports drawing tools, shapes, text placement, brush sizes, and canvas saving.
---
## ⚙️ Technologies
- Python
- Tkinter
---
## 🚀 Features
- Pencil tool
- Straight line tool
- Rectangle
- Circle
- Square
- Triangle
- Rhombus
- Brush sizes
- Color picker
- Text tool
- Ctrl+S canvas save
---
## ▶️ Run
```bash
python paint.py
